# RAG Document Assistant

## Project Description

This project is a RAG-based document assistant. It will allow users to upload PDF documents and ask questions based on those documents.

## Planned Features

- Upload PDF
- Extract text from PDF
- Clean text
- Split text into chunks
- Generate embeddings
- Store embeddings in vector database
- Ask questions
- Generate answers using document context

## Tech Stack

- Python
- Streamlit
- FAISS or ChromaDB
- LangChain
- LLM API
- GitHub
